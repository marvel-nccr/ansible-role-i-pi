# Ansible Role: marvel-nccr.i-pi

An ansible role that installs [i-PI](http://ipi-code.org/) on Ubuntu.

## Installation

`ansible-galaxy install marvel-nccr.i-pi`

## Role Variables

See `defaults/main.yml`

## Example Playbook

  - hosts: servers
    roles:
    - role: marvel-nccr.i-pi
